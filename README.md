# 🎬 Movie Recommendation System

A simple Movie Recommendation System built using Machine Learning.

This project recommends similar movies based on movie descriptions, genres, and taglines using TF-IDF and Cosine Similarity.

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Files

```text
movies.ipynb          # Main notebook
movies_metadata.xlsb  # Dataset
```

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Raghunandan-22/content-based-movie-recommender.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Run `movies.ipynb`.

## 🎯 Example

```python
recommend("Avatar", 5)
```

Output:

```text
Avatar 2
Thor: Ragnarok
The Inhabited Island
Moontrap: Target Earth
The Three Musketeers
```
